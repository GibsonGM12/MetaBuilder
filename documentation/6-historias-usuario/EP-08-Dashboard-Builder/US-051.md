## Historia de Usuario

**Como** administrador,

**Quiero** configurar un widget de tipo KPI Card,

**Para** mostrar una métrica con formato enriquecido (moneda, porcentaje) y una etiqueta descriptiva.

## Descripción

El KPI Card es similar al Stat Card pero con formato avanzado. Muestra un valor agregado con formato (currency, number, percentage), una etiqueta descriptiva personalizada y opcionalmente un prefijo/sufijo.

## Criterios de Aceptación

```gherkin
Scenario: KPI Card con formato moneda
  Given que configuro aggregation="sum", field="precio", format="currency"
  When se renderiza el widget
  Then muestra "$45,000.00" con formato de moneda

Scenario: KPI Card con etiqueta personalizada
  Given que configuro label="Ventas totales del mes"
  When se renderiza el widget
  Then muestra la etiqueta debajo del valor

Scenario: KPI Card con formato porcentaje
  Given que configuro aggregation="avg", field="descuento", format="percentage"
  When se renderiza el widget
  Then muestra "15.5%" con formato de porcentaje
```

## Notas Adicionales

- Config JSON: `{aggregation: "count"|"sum"|"avg", field: string|null, format: "number"|"currency"|"percentage", label: string, prefix: string, suffix: string}`
- Formato number: separador de miles
- Formato currency: "$" + separador de miles + 2 decimales
- Formato percentage: valor + "%"
