## Historia de Usuario

**Como** usuario,

**Quiero** ver una lista de todos los dashboards disponibles,

**Para** seleccionar cuál quiero consultar.

## Descripción

La página de listado de dashboards muestra todos los dashboards disponibles en formato de cards. Cada card muestra el nombre, descripción, cantidad de widgets y fecha de creación. El dashboard por defecto tiene un badge especial.

## Criterios de Aceptación

```gherkin
Scenario: Listar dashboards
  Given que existen 3 dashboards
  When accedo a /dashboards
  Then veo 3 cards con la información de cada dashboard

Scenario: Card muestra información relevante
  Given que un dashboard tiene nombre "Ventas Q1" y 5 widgets
  When veo la lista
  Then la card muestra "Ventas Q1", la descripción, "5 widgets" y la fecha

Scenario: Dashboard por defecto tiene badge
  Given que un dashboard está marcado como default
  When veo la lista
  Then ese dashboard muestra un badge "Por defecto"

Scenario: Navegar a dashboard
  Given que veo la lista de dashboards
  When hago clic en una card
  Then navego a /dashboards/:id

Scenario: Empty state
  Given que no hay dashboards creados
  When accedo a /dashboards
  Then veo un mensaje "No hay dashboards disponibles"
  And si soy Admin, veo un botón "Crear dashboard"
```

## Notas Adicionales

- Usar grid responsive de cards (3 columnas en desktop, 1 en mobile)
- Admin ve botón "Crear dashboard" que navega al designer
- Diseño con TailwindCSS consistente con el resto de la app
