## Historia de Usuario

**Como** administrador,

**Quiero** marcar un dashboard como "por defecto",

**Para** que los usuarios lo vean automáticamente al entrar al sistema.

## Descripción

Un dashboard puede ser marcado como default. Solo puede haber un dashboard por defecto a la vez. Cuando se marca uno, el anterior pierde el flag. La página de inicio puede redirigir al dashboard por defecto si existe.

## Criterios de Aceptación

```gherkin
Scenario: Marcar dashboard como default
  Given que tengo 3 dashboards sin default
  When marco uno como default
  Then ese dashboard tiene is_default=true
  And los demás permanecen con is_default=false

Scenario: Cambiar dashboard default
  Given que "Dashboard A" es el default
  When marco "Dashboard B" como default
  Then "Dashboard B" tiene is_default=true
  And "Dashboard A" cambia a is_default=false

Scenario: Home redirige a dashboard default
  Given que existe un dashboard por defecto
  When accedo a la página de inicio
  Then veo el dashboard por defecto renderizado
```

## Notas Adicionales

- Solo un dashboard puede ser default a la vez
- Al marcar uno como default, se debe desmarcar el anterior en una transacción
- La página Home puede mostrar el dashboard default en lugar del contenido actual
