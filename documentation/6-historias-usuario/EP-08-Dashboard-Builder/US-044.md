## Historia de Usuario

**Como** administrador,

**Quiero** configurar un widget de tipo Stat Card,

**Para** mostrar un valor numérico agregado (count, sum, avg) de una entidad.

## Descripción

El Stat Card es el widget más simple: muestra un número grande con una etiqueta. Por ejemplo, "150 Productos" o "Total: $45,000". La configuración permite elegir qué agregación aplicar y sobre qué campo.

## Criterios de Aceptación

```gherkin
Scenario: Stat Card muestra count de registros
  Given que configuro un Stat Card con aggregation="count" para entidad "Productos"
  When se renderiza el widget
  Then muestra el número total de registros de Productos

Scenario: Stat Card muestra sum de un campo
  Given que configuro un Stat Card con aggregation="sum" y field="precio"
  When se renderiza el widget
  Then muestra la suma del campo precio de todos los registros

Scenario: Stat Card con color e icono
  Given que configuro color="blue" e icon="package"
  When se renderiza el widget
  Then el fondo es azul y muestra el icono de paquete
```

## Notas Adicionales

- Config JSON: `{aggregation: "count"|"sum"|"avg", field: string|null, color: string, icon: string}`
- Si aggregation es "count", field puede ser null
- Colores soportados: blue, green, red, yellow, purple, indigo
- Iconos: usar lucide-react
