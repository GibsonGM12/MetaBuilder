## Historia de Usuario

**Como** usuario,

**Quiero** ver un dashboard con todos sus widgets renderizados,

**Para** consultar información consolidada de las entidades del sistema.

## Descripción

La vista de dashboard renderiza todos los widgets en sus posiciones configuradas. Cada widget carga sus datos de forma independiente desde la API. Los widgets muestran estados de carga, error y datos vacíos de forma individual (un widget con error no afecta a los demás).

## Criterios de Aceptación

```gherkin
Scenario: Dashboard renderiza todos los widgets
  Given que accedo a /dashboards/:id
  When el dashboard tiene 5 widgets configurados
  Then los 5 widgets se renderizan en sus posiciones correctas

Scenario: Widgets cargan datos independientemente
  Given que un dashboard tiene 3 widgets
  When uno de ellos falla al cargar datos
  Then los otros 2 widgets se muestran correctamente
  And el widget con error muestra un mensaje de error

Scenario: Widget muestra skeleton durante carga
  Given que accedo a un dashboard
  When los datos de un widget están cargando
  Then el widget muestra un skeleton/loading state

Scenario: Dashboard no es editable para usuario
  Given que soy un usuario con rol User
  When veo un dashboard
  Then los widgets no son arrastrables ni redimensionables
  And no veo botón de "Editar"
```

## Notas Adicionales

- Usar react-grid-layout en modo estático (no editable)
- Cada widget usa WidgetRenderer que renderiza el componente correcto según widget_type
- Error boundary por widget para aislar fallos
- Botón "Editar" visible solo para Admin
