## Historia de Usuario

**Como** administrador,

**Quiero** configurar un widget de tipo Recent List,

**Para** mostrar los últimos N registros de una entidad en formato de lista compacta.

## Descripción

El Recent List muestra los registros más recientes de una entidad en formato de lista vertical. Cada ítem muestra los campos seleccionados y la fecha de creación. Es útil para ver actividad reciente.

## Criterios de Aceptación

```gherkin
Scenario: Recent List muestra últimos registros
  Given que configuro limit=5 y fields=["nombre", "precio"]
  When se renderiza el widget
  Then muestra los 5 registros más recientes con nombre y precio

Scenario: Recent List ordena por fecha
  Given que hay 20 registros
  When se carga el widget con limit=5
  Then muestra los 5 más recientes ordenados por created_at DESC

Scenario: Recent List con campos seleccionados
  Given que una entidad tiene 8 campos
  When configuro fields=["nombre", "estado"]
  Then solo se muestran esos 2 campos por registro
```

## Notas Adicionales

- Config JSON: `{fields: string[], limit: number}`
- Limit por defecto: 5
- Ordenamiento fijo: created_at DESC
- Formato compacto: cada registro en una fila con campos separados
