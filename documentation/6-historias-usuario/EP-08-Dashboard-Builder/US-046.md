## Historia de Usuario

**Como** administrador,

**Quiero** configurar widgets de tipo gráfico (barras, pastel, líneas),

**Para** visualizar datos agregados y tendencias de las entidades de forma gráfica.

## Descripción

Los widgets de tipo Chart permiten visualizar datos agrupados. Soporta tres subtipos: BAR_CHART (barras), PIE_CHART (pastel) y LINE_CHART (líneas/series temporales). Cada uno agrupa datos por un campo y aplica una agregación.

## Criterios de Aceptación

```gherkin
Scenario: Bar Chart agrupa por campo de texto
  Given que configuro un BAR_CHART con groupBy="categoria" y aggregation="count"
  When se renderiza el widget
  Then muestra barras con una por cada valor único de categoría
  And la altura de cada barra es el count de registros con esa categoría

Scenario: Pie Chart muestra distribución
  Given que configuro un PIE_CHART con groupBy="estado" y aggregation="count"
  When se renderiza el widget
  Then muestra un gráfico de pastel con secciones por cada estado

Scenario: Line Chart muestra tendencia temporal
  Given que configuro un LINE_CHART con xAxis="created_at" y yAxis="monto" y interval="month"
  When se renderiza el widget
  Then muestra una línea con puntos por cada mes
  And el eje Y muestra la suma de monto por mes
```

## Notas Adicionales

- BAR_CHART config: `{groupBy: string, aggregation: "count"|"sum"|"avg", field: string|null}`
- PIE_CHART config: `{groupBy: string, aggregation: "count"|"sum"}`
- LINE_CHART config: `{xAxis: string, yAxis: string, aggregation: "sum"|"avg"|"count", interval: "day"|"week"|"month"}`
- Usar librería recharts para los gráficos
- Colores automáticos con paleta consistente
