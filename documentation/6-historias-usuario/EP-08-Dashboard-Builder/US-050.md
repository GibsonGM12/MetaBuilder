# US-050: Editar y eliminar dashboard

## Información General

| Campo | Valor |
|-------|-------|
| **ID** | US-050 |
| **Título** | Editar y eliminar dashboard |
| **Épica** | EP-08: Dashboard Builder |
| **Prioridad** | P1 (Alta) |
| **Estimación** | 45 min |
| **Dependencias** | US-042, US-047 |

## Historia de Usuario

**Como** administrador,

**Quiero** editar el nombre/descripción de un dashboard y poder eliminarlo,

**Para** mantener organizados los dashboards del sistema.

## Descripción

El administrador puede modificar los metadatos de un dashboard (nombre, descripción) y eliminar dashboards que ya no son necesarios. La eliminación es en cascada (elimina todos los widgets asociados).

## Criterios de Aceptación

```gherkin
Scenario: Editar nombre de dashboard
  Given que tengo un dashboard "Ventas Q1"
  When actualizo el nombre a "Ventas Q1 2026"
  Then el dashboard se actualiza exitosamente

Scenario: Eliminar dashboard
  Given que tengo un dashboard con 5 widgets
  When elimino el dashboard
  Then el dashboard y todos sus widgets se eliminan

Scenario: Confirmación antes de eliminar
  Given que hago clic en "Eliminar dashboard"
  When aparece el modal de confirmación
  Then debo confirmar para proceder con la eliminación

Scenario: Solo Admin puede editar/eliminar
  Given que soy un usuario con rol User
  When intento editar o eliminar un dashboard
  Then recibo un error 403
```

## Notas Adicionales

- DELETE en cascada: al eliminar dashboard se eliminan todos los dashboard_widgets
- Modal de confirmación para eliminación
- PUT /api/dashboards/:id para actualización
