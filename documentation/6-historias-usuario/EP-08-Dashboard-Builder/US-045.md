## Historia de Usuario

**Como** administrador,

**Quiero** configurar un widget de tipo Data Grid,

**Para** mostrar una tabla compacta con registros de una entidad dentro del dashboard.

## Descripción

El Data Grid muestra una tabla con columnas seleccionadas de la entidad, con paginación compacta. El admin elige qué columnas mostrar, cuántas filas por página y por qué campo ordenar.

## Criterios de Aceptación

```gherkin
Scenario: Data Grid muestra columnas seleccionadas
  Given que configuro un Data Grid con columns=["nombre", "precio"] para "Productos"
  When se renderiza el widget
  Then la tabla muestra solo las columnas Nombre y Precio

Scenario: Data Grid con paginación
  Given que configuro pageSize=5
  When hay 20 registros
  Then muestra 5 registros por página con controles de paginación compactos

Scenario: Data Grid con ordenamiento
  Given que configuro sortBy="created_at" y sortOrder="desc"
  When se cargan los datos
  Then los registros aparecen ordenados por fecha de creación descendente
```

## Notas Adicionales

- Config JSON: `{columns: string[], pageSize: number, sortBy: string, sortOrder: "asc"|"desc"}`
- Las columnas usan el `name` del campo, se muestra el `display_name` como header
- pageSize por defecto: 5
- Estilo compacto para caber dentro de un widget
