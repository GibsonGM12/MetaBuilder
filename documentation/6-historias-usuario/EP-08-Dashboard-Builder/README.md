# EP-08: Dashboard Builder

## Descripción

Épica que implementa un sistema completo de dashboards configurables donde los administradores pueden diseñar paneles de visualización con widgets (cards, gráficos, grids) que consultan datos de las entidades dinámicas, y los usuarios pueden consumir estos dashboards.
