## Historia de Usuario

**Como** administrador,

**Quiero** posicionar y redimensionar widgets arrastrándolos en un canvas,

**Para** diseñar el layout del dashboard de forma visual e intuitiva.

## Descripción

El Dashboard Designer usa un grid de 12 columnas donde los widgets se pueden arrastrar y redimensionar. Cada tipo de widget tiene tamaños mínimos. La posición y tamaño se guardan como `{x, y, w, h}` en la base de datos.

## Criterios de Aceptación

```gherkin
Scenario: Arrastrar widget a nueva posición
  Given que estoy en el Dashboard Designer
  When arrastro un widget de la posición (0,0) a (4,2)
  Then el widget se mueve visualmente
  And la posición se actualiza en el estado local

Scenario: Redimensionar widget
  Given que tengo un widget de tamaño 4x3
  When lo redimensiono a 6x4
  Then el widget cambia de tamaño visualmente
  And respeta el tamaño mínimo del tipo de widget

Scenario: Widgets no se superponen
  Given que tengo dos widgets
  When arrastro uno sobre otro
  Then el segundo widget se desplaza automáticamente

Scenario: Guardar layout
  Given que he modificado posiciones de widgets
  When hago clic en "Guardar"
  Then las posiciones se persisten en la base de datos
```

## Notas Adicionales

- Usar `react-grid-layout` con `cols: 12`, `rowHeight: 80`
- Tamaños mínimos: STAT_CARD(2x2), KPI_CARD(2x2), DATA_GRID(4x3), CHART(4x3), RECENT_LIST(3x3)
- En modo view (usuario normal): `isDraggable: false`, `isResizable: false`
- En modo designer (admin): `isDraggable: true`, `isResizable: true`
