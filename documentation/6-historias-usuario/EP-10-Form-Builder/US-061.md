## Historia de Usuario

**Como** administrador,

**Quiero** agregar una sección FIELDS a un formulario para capturar campos simples de una entidad,

**Para** que los usuarios puedan ingresar datos básicos (texto, número, fecha, etc.) en el formulario.

## Descripción

El administrador puede agregar una sección de tipo FIELDS a un formulario. Esta sección muestra campos simples (TEXT, NUMBER, DATE, BOOLEAN, etc.) de una entidad. El admin selecciona qué campos incluir y opcionalmente el orden de visualización. La sección puede referirse a la entidad principal o a otra entidad relacionada.

## Criterios de Aceptación

```gherkin
Scenario: Admin agrega sección FIELDS exitosamente
  Given que tengo un formulario creado con entidad principal "Ventas"
  When agrego una sección FIELDS con entity_id="Ventas" y fields=["fecha", "status", "observaciones"]
  Then la sección se guarda en sections_config del formulario
  And la sección tiene type="FIELDS", entity_id, field_ids o field_names

Scenario: Admin selecciona campos de la entidad
  Given que estoy agregando una sección FIELDS
  When selecciono la entidad "Ventas"
  Then veo la lista de campos disponibles (excluyendo RELATION si aplica)
  And puedo marcar/desmarcar campos para incluir
  And puedo reordenar el orden de los campos

Scenario: Sección FIELDS muestra controles según tipo de campo
  Given que la sección incluye campos TEXT, NUMBER, DATE
  When el usuario llena el formulario
  Then ve input de texto para TEXT
  And ve input numérico para NUMBER
  And ve date picker para DATE

Scenario: Campos requeridos muestran validación
  Given que un campo tiene required=true en la entidad
  When el usuario intenta enviar sin llenar ese campo
  Then recibe mensaje de validación
  And el formulario no se envía
```

## Notas Adicionales

- La sección FIELDS se almacena en sections_config como objeto con: type, entity_id, field_ids, order
- Reutilizar componentes de formulario existentes del CRUD dinámico
- Campos de tipo RELATION no se incluyen en FIELDS; usar sección LOOKUP (US-062)
