# US-067: Listar formularios disponibles

## Información General

| Campo | Valor |
|-------|-------|
| **ID** | US-067 |
| **Título** | Listar formularios disponibles |
| **Épica** | EP-10: Form Builder |
| **Prioridad** | P0 (Crítica) |
| **Estimación** | 30 min |
| **Dependencias** | US-060 |

## Historia de Usuario

**Como** usuario,

**Quiero** ver todos los formularios disponibles en una lista,

**Para** poder seleccionar el formulario que necesito llenar.

## Descripción

El usuario puede ver una lista de todos los formularios creados por los administradores. La lista muestra nombre, descripción y entidad principal. Al hacer clic en un formulario, se abre para llenarlo. La lista está accesible desde el sidebar para todos los usuarios autenticados.

## Criterios de Aceptación

```gherkin
Scenario: Usuario ve lista de formularios
  Given que soy un usuario autenticado
  When navego a "Formularios" en el sidebar
  Then veo la lista de formularios disponibles
  And cada item muestra nombre, descripción (truncada), entidad principal
  And puedo hacer clic para abrir el formulario

Scenario: Lista vacía muestra empty state
  Given que no hay formularios creados
  When navego a la lista de formularios
  Then veo mensaje "No hay formularios disponibles"
  And si soy Admin, veo botón para crear formulario

Scenario: GET /api/forms retorna formularios
  Given que existen 3 formularios
  When envío GET /api/forms
  Then recibo array con los 3 formularios
  And cada uno tiene id, name, description, primary_entity_id, sections_config
  And la respuesta está paginada si hay muchos

Scenario: Usuario sin rol Admin ve formularios
  Given que soy User (no Admin)
  When accedo a la lista de formularios
  Then veo los formularios disponibles para llenar
  And no veo opción de "Diseñar" o "Crear" (solo Admin)
```

## Notas Adicionales

- Endpoint: GET /api/forms (lista) y GET /api/forms/:id (detalle con sections_config)
- La lista puede ser cards o tabla según diseño
- Filtrar por entidad principal si hay muchos formularios (opcional)
