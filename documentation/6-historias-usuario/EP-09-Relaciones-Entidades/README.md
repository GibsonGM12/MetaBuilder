# EP-09: Relaciones entre Entidades

## Descripción

Épica que implementa la capacidad de definir relaciones entre entidades del sistema, permitiendo que los datos de una entidad referencien registros de otra entidad. Los administradores pueden crear relaciones MANY_TO_ONE y campos tipo RELATION, mientras que los usuarios pueden seleccionar registros relacionados mediante dropdowns con búsqueda en los formularios CRUD.
